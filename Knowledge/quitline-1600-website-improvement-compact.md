# Quitline 1600 Website Improvement Brief — Compact

## Project
ปรับปรุงเว็บไซต์ **Quitline 1600** ให้เป็น **Digital Quit Support Platform** สำหรับเชื่อมผู้ใช้งานเข้าสู่บริการเลิกบุหรี่ผ่าน **โทร 1600, LINE OA / QuitBot น้องวันใหม่, แบบประเมิน, คลังความรู้ และระบบติดตามผล**

## Current Problems
- **Mobile Experience:** ใช้งานบนมือถือยังไม่ลื่นไหล ปุ่ม/เมนูใช้งานยาก
- **Information Overload:** หน้าแรกข้อมูลแน่น ขาด Visual Hierarchy ผู้ใช้หา CTA ไม่เจอ
- **Outdated UI:** รูปแบบ สี และเลย์เอาต์ดูเก่า ไม่ดึงดูดกลุ่มผู้ใช้รุ่นใหม่

## Objectives
1. ปรับเว็บไซต์ให้ทันสมัย ใช้งานง่าย และออกแบบแบบ **Mobile First**
2. ทำให้เว็บไซต์เป็นช่องทางหลักสู่ **Quitline 1600** และ **QuitBot น้องวันใหม่**
3. จัดระบบความรู้เรื่องการเลิกบุหรี่ให้เข้าถึงง่ายและค้นหาได้สะดวก
4. เพิ่มเครื่องมือดิจิทัลเพื่อสร้างแรงจูงใจในการเลิกบุหรี่
5. มีระบบวิเคราะห์ข้อมูลผู้ใช้งานและผลลัพธ์การสื่อสาร
6. รองรับ **SEO / SEM / Digital Marketing**
7. รองรับ **PDPA** และการจัดการความยินยอม

## Target Users
### Primary
- ผู้สูบบุหรี่ที่ต้องการเลิก
- ผู้ที่กำลังพิจารณาเลิกบุหรี่
- ผู้ใช้บุหรี่ไฟฟ้า
- ผู้ที่เคยพยายามเลิกแต่ไม่สำเร็จ

### Secondary
- ครอบครัวและคนใกล้ชิดของผู้สูบบุหรี่
- บุคลากรทางการแพทย์ / สาธารณสุข
- โรงพยาบาลและหน่วยบริการสุขภาพ
- ประชาชนทั่วไปที่ต้องการข้อมูลการเลิกบุหรี่

## Key User Journeys
### 1. ผู้ต้องการเลิกบุหรี่
ผู้ใช้เห็นสื่อจาก Facebook / TikTok / Google / QR Code → เข้าเว็บไซต์ → เลือกช่องทางรับบริการ

**ในเวลาทำการ**
- โทร 1600 เพื่อรับคำปรึกษา
- Add LINE / เข้าร่วมโปรแกรมผ่าน QuitBot
- รับคำปรึกษา → เข้าสู่โปรแกรม → ติดตามผลต่อเนื่อง

**นอกเวลาทำการ**
- คุยกับ **น้องวันใหม่ Chatbot**
- กรอกฟอร์ม **U Quit** ให้เจ้าหน้าที่โทรกลับ
- รับคำแนะนำเบื้องต้น → ประเมินความพร้อม → ส่งต่อเจ้าหน้าที่ → ติดตามผล

### 2. ผู้ค้นหาข้อมูล
Google / Social → เข้าเว็บไซต์ → อ่านบทความ / วิดีโอ / Infographic / เครื่องมือช่วยเลิกบุหรี่ → ทำ Quiz / แบบประเมิน / เกม → แชร์ต่อ หรือเปลี่ยนเข้าสู่ Journey โทร/LINE ได้ทันที

## Design Principles
- Mobile First Design
- User-Centered Design
- เข้าถึงบริการได้ภายใน **1–2 คลิก**
- CTA ชัดเจนทุกหน้า
- ใช้ภาษาง่าย เข้าใจไว
- ลดขั้นตอนการเข้าถึงบริการ
- รองรับผู้สูงอายุและผู้มีทักษะดิจิทัลจำกัด
- สื่อสารเชิงบวกและสร้างแรงจูงใจ
- เชื่อมต่อ **โทร 1600 / LINE OA / QuitBot** อย่างไร้รอยต่อ
- แนวทางอ้างอิง: **Quitline Australia** — เน้นช่วยเหลือผู้ใช้มากกว่าประชาสัมพันธ์องค์กร
